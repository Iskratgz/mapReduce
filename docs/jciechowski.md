# Map reduce
### *Jakub Ciechowski*

Obliczamy sumę trzęsień ziemi dla danego regionu

## Funkcja map
```js
var map = function() {
	emit(this.Region, 1);
};
```

## Funkcja reduce
```js
var reduce = function(key, values) {
	var sum = 0;
	values.forEach(function(v) {
		sum += 1
	});
	return {
		count: sum
	};
};
```

## Map Reduce kolekcji
```js
db.earthquakes.mapReduce(map, reduce, {
	out: "earthquakes_sum"
});
```

## Wyświetlenie rezultatu
```js
db.earthquakes_sum.find({});
```
